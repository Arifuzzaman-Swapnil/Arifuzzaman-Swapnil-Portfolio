# Arifuzzaman Swapnil - Portfolio

A modern, responsive personal portfolio website for **Md Arifuzzaman Swapnil** — Backend AI Engineer & Full Stack Python Developer.

**Live Preview:** [arifuzzaman-swapnil-portfolio](https://arifuzzaman-swapnil.github.io/Arifuzzaman-Swapnil-Portfolio/)

## Tech Stack

- **React 18** with TypeScript
- **Vite** — fast build tool and dev server
- **Tailwind CSS** — utility-first styling
- **shadcn/ui** — accessible UI components built on Radix UI
- **Framer Motion** — smooth scroll and entrance animations
- **React Router** — client-side routing

## Features

- Responsive design with mobile navigation
- Dark theme with cyan/teal accent gradients
- Animated sections on scroll (Hero, About, Education, Experience, Projects, Skills, Awards)
- 6 showcased AI/ML projects with GitHub links
- 7 categorized skill groups
- Awards, certifications, and research interests
- Custom typography (Space Grotesk + JetBrains Mono)

## Getting Started

```bash
# Clone the repository
git clone https://github.com/Arifuzzaman-Swapnil/Arifuzzaman-Swapnil-Portfolio.git

# Navigate to the project
cd Arifuzzaman-Swapnil-Portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

## Available Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Production build |
| `npm run preview` | Preview production build |
| `npm run lint` | Run ESLint |
| `npm run test` | Run tests |
